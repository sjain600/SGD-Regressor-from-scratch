# Introduction

Building a model from scratch is one thing, but implementing the code into action is where real learning happens. You get a chance to test how your model is performing, identify why it is failing, and evaluate it against different metrics to gauge its performance.

In this folder, you will find the models that have been tested on the datasets and refined based on their mistakes. The starting point (base model) would be building the model that won the Netflix Prize, as described in Koren's research paper. I will let my curiosity dive into more advanced, cutting-edge techniques, including learning hard maths, building intuition, and testing them across different datasets. I will upload my findings here in this folder.

As I go down the rabbit hole, I hope to build something innovative. Stay tuned for something exciting!
