# Introduction

Building a model from scratch is one thing, but implementing the code into action is where real learning happens. You get a chance to test how your model is performing, identify why it is failing, and evaluate it against different metrics to gauge its performance.

In this folder, you will find the models that have been tested and refined based on their mistakes. The model I am building is inspired by the Netflix Price Winner from the Koren research paper, which is the starting point (base model). I will let my curiosity dive into more advanced techniques, including learning maths, building an intuition, and testing them against different datasets.

As I go down the rabbit hole, I hope to build something innovative. Stay tuned for something exciting!
